## 🚀 About

**JARVIS** is a futuristic, modular AI assistant built specifically for **Android + Termux**. It combines powerful AI models, Android automation, and a sleek Iron Man–inspired interface to deliver a smart, extensible, and developer-friendly assistant.

---

## ✨ Features

### 🧠 Cognitive Engine
- 🤖 LLM Integration (GLM 5.2 & OpenRouter)
- 🧩 Context-aware Memory (SQLite)
- 💡 Intelligent Reasoning & Task Planning

### 📱 Android Automation
- 📶 Wi-Fi & Bluetooth Control
- 🔦 Flashlight & Volume Management
- 📍 GPS & Sensor Monitoring
- ⚡ Background Automation Workflows

### 🎨 Futuristic Interface
- 🌌 Iron Man–Inspired UI
- 💙 Dark Blue Holographic Theme
- 🌀 Three.js Powered 3D Visuals
- 🎙️ Real-time Voice Wave Animation

## 🚀 Installation

```bash
pkg update && pkg upgrade -y
pkg install git python termux-api -y

git clone https://github.com/rudraksha092-beep/Android-Jarvis-.git
cd Android-Jarvis-

bash scripts/install.sh
```
